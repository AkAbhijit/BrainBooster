HTML Tables

What is an HTML Table?

HTML Tables are powerful tools for organizing and displaying data in a structured format. They create a grid-like structure of rows and columns, making it easy to present information in a clear and organized manner. While tables were historically used for page layouts, modern web development best practices reserve tables specifically for tabular data.

Key benefits of HTML tables include:

  • Organized presentation of data
  • Built-in accessibility features
  • Easy comparison of related information
  • Support for complex data structures

Table Syntax and Structure

HTML tables are built using a combination of elements that work together to create the table structure. Each element has a specific role in defining the table's layout and content. 

<table>
    <caption>Popular Books</caption>
    <thead>
        <tr>
            <th>Book Name</th>
            <th>Author</th>
            <th>Genre</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>The Book Thief</td>
            <td>Markus Zusak</td>
            <td>Historical Fiction</td>
        </tr>
    </tbody>
</table>

Advanced Table Features

HTML tables support several advanced features that enhance their functionality and appearance:

Feature Description Example
Spanning Cells Merge cells across rows or columns colspan="2"
Table Sections Organize table into header, body, and footer <thead>, <tbody>, <tfoot>
Column Groups Define properties for entire columns <colgroup>, <col>

Quick Reference

Tag Description Attributes
<table> Defines a table border, cellpadding, cellspacing
<tr> Defines a table row align, valign
<th> Defines a header cell colspan, rowspan, scope
<td> Defines a data cell colspan, rowspan
<caption> Defines a table caption align
