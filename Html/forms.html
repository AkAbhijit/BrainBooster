HTML Forms

Introduction to HTML Forms

HTML Forms are powerful tools for collecting user input through various interactive controls. They serve as containers for different input elements, enabling rich user interactions and data collection.

Basic Syntax

<form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
    
    <button type="submit">Submit</button>
</form>

Form Elements

HTML forms comprise several elements, each serving a unique purpose in capturing user input.

Element Description
<label> Defines labels for form elements
<input> Captures various input types
<button> Creates a clickable button
<select> Generates a drop-down list
<textarea> Captures long text inputs
<fieldset> Groups related form elements

Input Types

HTML forms support various input types to collect specific data from users.

Input Type Description
type="text" Single-line text input
type="password" Password input field
type="email" Email address input with validation
type="number" Number input with min/max
type="checkbox" Multiple selection checkbox
type="radio" Single selection radio button
