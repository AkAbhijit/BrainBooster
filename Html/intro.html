What is HTML?

HTML (HyperText Markup Language) is the foundation of web development. It's a markup language that defines the structure and content of web pages through a series of elements and tags.

Basic Structure

Every HTML document follows a standard structure that includes several key elements:

<!DOCTYPE html>

Declares that this is an HTML5 document

<html>

The root element of an HTML page

<head>

Contains meta information about the document

<body>

Defines the document's body content

HTML Elements

HTML elements are the building blocks of web pages. Here are some commonly used elements:

<h1> to <h6>

Headings of different levels

<p>

Paragraph text

<a>

Creates hyperlinks

<img>

Embeds images

<ul> and <ol>

Unordered and ordered lists

<div>

Container for flow content

Example Code

Here's a simple HTML document structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first HTML page!</p>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>
    <a href="https://example.com">Visit Example.com</a>
</body>
</html>
